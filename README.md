# WikiBot 🤖

A Telegram bot that fetches Wikipedia articles with multi-language support.

## Features
- Search Wikipedia in 3 languages (Uz, Ru, En)
- Random article generation
- Disambiguation handling
- User language preferences

## Setup
1. Clone repo
2. `pip install -r requirements.txt`
3. Create `.env` with BOT_TOKEN
4. `python bot.py`

## Commands
- `/start` - Welcome
- `/wiki <query>` - Search
- `/lang` - Change language
